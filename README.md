# Calculadora-Intalaciones-AA
programa para calcular rapidamente una instalacion secilla de AA
Alcance del Proyecto y Requisitos Funcionales
Alcance: El sistema permite a un técnico o cliente obtener un presupuesto estimado para la instalación de equipos de aire acondicionado, automatizando el cálculo de materiales y mano de obra, y filtrando trabajos de alta complejidad que requieren inspección presencial.
Requisitos Funcionales (Lo que hace el programa):
El sistema debe permitir seleccionar la capacidad del equipo (frigorías).
El sistema debe validar si el trabajo requiere altura o si el equipo es mayor a 5000 frigorías para solicitar visita técnica obligatoria.
El sistema debe calcular costos adicionales por metros excedentes de cañería (más de 3m).
El sistema debe permitir sumar opcionales como el "Kit de Materiales" y la "Instalación Eléctrica".
